Data Entry Guidelines

Please Follow These Guidelines:

  • Fill in all required fields carefully to ensure accurate data collection.
  • You can come back to change any data at any time by navigating to the previous sections.
  • Images selected should be in JPEG format for optimal quality.
  • Ensure that the dimensions of the images are appropriate for the report (e.g., 100px width and 60px height).
  • Review your entries before generating the PDF to avoid any mistakes.
  • If you encounter any issues, please refer to the help section or contact support.
